# ElectroFix-AI

Sistema SaaS para gestión de servicio técnico de electrodomésticos, construido con Laravel, Blade y Bootstrap.

## Estado actual
- Autenticación por sesión
- Multi-tenant por empresa
- Dashboards segmentados por rol (`worker`, `admin`, `developer`)
- Permisos delegables por admin para workers (`Facturación` e `Inventario`)
- Módulos principales con lógica backend:
  - Órdenes
  - Clientes
  - Equipos

## Stack
- Laravel 12
- Blade
- Bootstrap 5
- MySQL/MariaDB
